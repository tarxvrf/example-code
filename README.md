## Untuk File .env 

kalian bisa tulis 
```bash
EMAIL_SERVER_HOST="smtp.mail.com"                                    bisa pakai email provider calian sperte yahoo,hotmail,gmail
SMTP_USER="youremail"                  
SMTP_PASS="your pass app"                                            bisa kalian dapatkan saat setting di google developer console dibagian APP password
EMAIL_SERVER_FROM="youremail"
NEXTAUTH_URL="yourdomain"
NEXTAUTH_SECRET="yoursecret"
DATABASE_URL="postgresql://youruser:yourpass@localhost:5432/yourdb?schema=public"                jika menggunakan dbPostgresql
DATABASE_URL="mysql://youruser:yourpass@localhost:3306/yourdb"                                  jika menggunakan db MYSQL
GOOGLE_CLIENT_ID="yourgoogle clientid"                                                          bisa kalian dapatkan saat setting di google developer console dibagian api credential OAUTH2.0
GOOGLE_CLIENT_SECRET="your google client secret"                                              bisa kalian dapatkan saat setting di google developer console dibagian api credential OAUTH2.0

```
